# 兩輪差速 AGV 路徑規劃模擬器

一個基於瀏覽器的即時模擬平台，用於測試與驗證兩輪差速驅動 AGV（無人搬運車）在複雜環境下的移動路徑規劃與動態避障能力。

## 功能特色

- 🤖 **真實差速運動模型** — 精確模擬左右輪速差驅動的機器人運動學
- 🗺️ **50m × 50m 大型地圖** — 帶座標格線的笛卡兒座標系地圖
- 🧭 **自動導航** — AGV 自動追蹤目標點 B，計算最優路徑
- 🚧 **動態避障** — 即時偵測障礙物並主動修正路徑（支援 2m 寬通道）
- ⚡ **高速模擬** — 最高 3000 RPM / 600 mm/s，支援 15 倍速預覽
- 🖱️ **直觀操作** — 左鍵放置障礙物、右鍵設定目標點
- 🔵 **路徑軌跡** — AGV 移動時留下漸淡藍色軌跡

## 操作說明

| 操作 | 功能 |
|------|------|
| 左鍵點擊地圖 | 放置障礙物 |
| 右鍵點擊地圖 | 設定目標點 B |
| Start / Pause / Reset | 控制模擬流程 |
| 障礙物形狀切換 | 正方形 / 圓形（各 1m） |
| 速度倍率滑桿 | 1× ~ 15× 加速預覽 |
| 滑鼠懸停地圖 | 即時顯示游標世界座標 |

## 鍵盤快捷鍵

| 按鍵 | 功能 |
|------|------|
| `Space` | 開始 / 暫停模擬 |
| `R` | 重置模擬（保留障礙物與目標） |
| `Escape` | 清除目標點 B |

## 技術規格

- **車體尺寸**：1m × 1m 正方形
- **最高速度**：600 mm/s（3000 RPM）
- **地圖大小**：50m × 50m
- **平台**：純 HTML5 Canvas + JavaScript（無框架依賴）
